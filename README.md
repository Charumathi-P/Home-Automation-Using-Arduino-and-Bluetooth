# Home Automation Using Arduino and Bluetooth

## Description

Home Automation Using Arduino and Bluetooth is an IoT-based smart home system that enables wireless control of household appliances through a Bluetooth-enabled mobile application. The system uses an Arduino Uno, HC-05 Bluetooth module, and relay modules to provide convenient, reliable, and cost-effective automation within Bluetooth range.

## Features

* Wireless appliance control using Bluetooth
* Mobile application-based operation
* Real-time switching through relay modules
* Easy installation and user-friendly interface
* Reliable short-range communication
* Low-cost and energy-efficient automation solution

## Components Used

* Arduino Uno
* HC-05 Bluetooth Module
* Relay Module
* Mobile Application
* Power Supply
* Connecting Wires
* Breadboard/PCB
